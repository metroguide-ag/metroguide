# MetroGuide — Deploy Guide
## GitHub + Vercel · 15 минут от нуля до живого сайта

---

## Структура файлов (у тебя два файла)

```
metroguide/
├── index.html     ← сайт (не трогай)
└── content.js     ← ВСЕ тексты здесь (редактируй)
```

---

## Шаг 1 — GitHub репозиторий

1. Зайди на **github.com** → Sign up (если нет аккаунта)
2. Нажми **"New repository"** (зелёная кнопка вверху)
3. Название: `metroguide` (или любое)
4. Visibility: **Public** ← важно для бесплатного Vercel
5. Нажми **"Create repository"**
6. На следующей странице нажми **"uploading an existing file"**
7. Перетащи оба файла (`index.html` и `content.js`) в окно
8. Нажми **"Commit changes"**

✅ Готово — файлы в GitHub.

---

## Шаг 2 — Vercel деплой

1. Зайди на **vercel.com** → Continue with GitHub
2. Нажми **"Add New Project"**
3. Найди репозиторий `metroguide` → нажми **"Import"**
4. Настройки оставь дефолтными (Framework: Other)
5. Нажми **"Deploy"**

⏱ Через 30–60 секунд сайт живой на адресе типа:
`metroguide.vercel.app`

---

## Шаг 3 — Свой домен (опционально)

Если есть домен (например `metroguide.io`):

1. В Vercel → твой проект → **Settings → Domains**
2. Добавь домен → Vercel покажет два DNS-записи
3. Зайди к регистратору домена (Namecheap / GoDaddy / etc.)
4. Добавь эти записи в DNS
5. Через 5–30 минут сайт доступен по своему домену

## Как редактировать контент

### Способ 1 — Прямо в браузере (GitHub)

1. Зайди на `github.com/ТВО_ИМЯ/metroguide`
2. Кликни на файл **`content.js`**
3. Нажми карандаш ✏️ (Edit this file)
4. Измени нужный текст
5. Нажми **"Commit changes"** → **"Commit directly to main"**

⏱ Vercel автоматически задеплоит изменения за ~30 секунд.

### Что редактировать — примеры

**Изменить описание кейса Bytemine:**
```js
// Найди в content.js блок cases, кейс "Bytemine"
desc: "Твой новый текст здесь.",
```

**Добавить новый кейс:**
```js
{
  id: "новый-кейс",
  tag: "SaaS · Landing Page · 8 screens",
  title: "Название",
  desc: "Описание проекта.",
  wide: false,
  color: "#your-color",
  icon: "◐",
},
```

**Изменить цену сервиса:**
```js
price: "From $2,000 · 3 weeks",
```

**Изменить email:**
```js
email: "новый@email.com",
```

## Частые вопросы

**Сломался сайт после правки?**
→ Зайди в GitHub → history → нажми на предыдущий commit → "Revert"

**Хочу добавить скриншоты в карточки кейсов?**
→ Залей картинки в папку `/images/` в репозитории
→ В `content.js` добавь поле `image: "/images/bytemine.jpg"` в кейс
→ Скажи мне — добавлю поддержку изображений в index.html

**Как проверить сайт локально перед деплоем?**
→ Просто открой `index.html` в браузере (двойной клик)
→ Убедись что `content.js` лежит в той же папке
